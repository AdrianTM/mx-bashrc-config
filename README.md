# mx-bashrc-config
A GUI tool written in Qt to modify the bashrc

## Bulding and Installing

* run ./build.sh
* the following step should have created an executable
* run ./install-themes.sh to install themes

## Theme Syntax

Is changing

(BOLD(T or F):RRR,GGG,BBB:CHARS)

## Dependencies

Qt5 Core

Qt5 GUI

Qt5 Widgets
